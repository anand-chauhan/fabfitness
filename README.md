💪 Fab Fitness – Gym & BMI Calculator Website

A modern and responsive Fitness Website that helps users explore gym services, fitness programs, and calculate their Body Mass Index (BMI) to track their health.

🔗 Live Demo: https://anand-chauhan.github.io/fabfitness/

🌟 About the Project

Fab Fitness is a fitness-focused web application designed to promote a healthy lifestyle. It provides information about gym services and includes a BMI Calculator to help users understand their fitness level.

✨ Features

🏋️ Gym & fitness services showcase

📊 BMI Calculator (Height & Weight based)

📱 Fully responsive design

🎨 Clean and modern UI

⚡ Fast and interactive experience

🛠️ Tech Stack

HTML – Structure

CSS – Styling & layout

JavaScript – Logic and calculations
